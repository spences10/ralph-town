# @ralph-town/core

Core library for Ralph Loop orchestration - runtimes, orchestrator, and types.

## Installation

```bash
npm install @ralph-town/core
```

## Usage

```typescript
import { orchestrate, run_ralph_loop } from '@ralph-town/core';
import { create_runtime } from '@ralph-town/core';
```
